This topic introduces the building blocks of networking on AWS. Work through the lessons below in order — each one is a self-contained interactive page. By the end you should be able to design a secure, well-segmented VPC from scratch.

You will learn:

  • What a VPC is and why it matters
  • Subnets, route tables, and CIDR planning
  • Internet, NAT, and gateway endpoints
  • Security groups vs. network ACLs
